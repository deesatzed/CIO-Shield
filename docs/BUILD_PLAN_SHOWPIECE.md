# Build Plan Showpiece

## North Star
Intent-preserving assistance that users trust because it is predictable, reversible, and auditable.

## Milestones
1. Trust shell: protected mode, panic toggle, ledger.
2. Suggest-only UX: boundary+idle trigger, accept/dismiss, budget/cooldown.
3. Undo + negative learning: anti-nag feedback loop.
4. Apple FM selector-only arbiter (on-chip, core for gray-zone decisions).
5. Proof report and health card for every session.

## Not-autocorrect proof points
- selector from known candidates, not open generation.
- default suggest-only, not silent rewrite.
- explicit do-nothing in risky/unknown contexts.
- user-visible ledger and local-only proof metrics.
