CIO-II

Intent-preserving typing assistance for macOS. Suggest-only by default, hard privacy blocks, perfect undo, and an optional internal Apple FM arbiter for ambiguous gray-zone corrections.

Built for trust, reversibility, and auditable data minimization.

Main Benefit of Apple FM

  • Improves ambiguous-case precision when deterministic confidence is borderline.
  • Can only select from known local candidates or return do nothing.
  • Reduces wrong interventions without turning CIO-II into freeform autocomplete.
  • Keeps core trust model: local-first, reversible, and safety-gated.

Why It Is Not Autocorrect

  • Default mode is suggest-only, not silent rewrite.
  • Unknown/code/terminal profiles fail safe to do nothing.
  • Optional FM path can only pick known candidate IDs or do nothing.
  • Every applied change is reversible via undo records and hotkey.

Privacy + Security Posture

  • Protected Mode hard-blocks sensitive fields and excluded apps.
  • Panic hotkey pauses capture and interventions instantly.
  • Ledger stores minimized events; no raw keystroke stream.
  • Core runtime path is local-only with no outbound network dependency.

Quick Start

git clone https://github.com/deesatzed/CIO-II.git
cd CIO-II
uv venv .venv
source .venv/bin/activate
uv pip install -e ".[dev,mac]"
./run_demo.sh
